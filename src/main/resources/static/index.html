Recommended usage flow

  1. Start the application. It runs on port 8090 unless you change it here.
  2. Open the generated Swagger URL or bookmarklet details below.
  3. Enter the Xray ticket and target page before you start documenting the flow.
  4. Create one step card per action. Use the title, action, expected result, test data, and notes textboxes.
  5. Use the bookmarklet on the target page to launch the selector picker script from the configured port.
  6. After capturing screenshots, upload them here and download renamed copies that include the Xray ticket and step number.
  7. Export the full session JSON to keep every input value in one file.

This page stores the current session in browser local storage so a page refresh does not wipe your entries.
