No browser extension required

Launch the UI Picker without installing Chrome or Edge extensions

If your browser blocks extension installation, use one of the fallbacks below. Both options load the same selector-picker.js script from this server, so the picker behavior stays the same.

Option 1: Bookmarklet

Drag the launcher to the bookmarks bar, then click it on any page where you want to inspect elements.

UI Picker Bookmarklet

Option 2: DevTools console

Paste the loader into the browser console when bookmarks are disabled by policy.

Why this works

                That means locator generation, ranked selectors, Cypress/Playwright snippets, and DOM inspection all remain unchanged.
